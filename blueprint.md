# Project Blueprint

## Overview
A simple framework-less web project using HTML, CSS, and JavaScript, following modern web standards (Baseline).

## Project Details
- **Entry Point:** `index.html`
- **Styling:** `style.css`
- **Logic:** `main.js`
- **Guidelines:** `GEMINI.md`

## Features & Styles
- Basic HTML5 structure.
- External CSS and JS linkage.
- Modern web development guidelines via `GEMINI.md`.
- **Lotto Number Generator:** Generates 5 sets of 6 unique numbers (1-45) with a premium UI.
